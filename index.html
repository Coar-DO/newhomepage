architecture

    통합환경 아키텍처

    Cloud-Native환경에서의 Devops, 가상화, MSA, CI/CD 구현

    주요 기능

    Kubernetes

    쿠버네티스에서는 가상화 컨테이너 환경에서 애플리케이션을 관리합니다. 여러가지 애플리케이션들은 서비스 타입으로 배포되어집니다.

    noVNC

    전체적인 시스템을 Web GUI 환경으로 체험할 수 있습니다. Centos 환경에서 CKA LAB환경이 완벽하게 구현되어있습니다.

    Grafana

    시스템의 상태를 모니터링하여 보여줍니다. 특정 애플리케이션을 관리하거나 전체 시스템 사용량을 관리할 때 사용합니다.

    Jenkins

    CI/CD의 대표적인 관리 도구로 Git을 통해 Kubernetes로의 배포작업을 주로합니다. SSH 연결을 기반으로 하며 여러 가지 플러그인을 지원합니다.

    CKA

    Cka 환경을 최대한으로 제공합니다. 17문제의 예상문제와 Lab환경을 체험해보시길 원하신다면 상단의 LAB 웹 환경과 CKA 예상문제를 눌러 문제를 풀어주세요. 제한시간은 2시간이며 총 66점이상을 맞아야 통과할 수 있습니다. 모든 문제를 푼 이후에는 자동화된 쉘 스크립트를 실행하여 점수를 얻어보세요! 좋은 결과가 함께하길 바랍니다. (문제는 홈페이지내 CKA 예상문제에서 찾을 수 있습니다.)
